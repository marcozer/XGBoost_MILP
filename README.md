# Purpose 

XGBoost model for prediction of Best Performers after Laparoscopic Distal Pancreatectomy trained on 55 French centers. The composite outcome was defined by was defined at 90 days as absence of mortality, severe complications, CR-POPF, reoperation, readmission, and LOS ≤75th percentile, while BP was IO with LOS ≤25th percentile.

# Deployment 

Model has been deployed on [streamlit platform](https://xgboostmilp.streamlit.app/)
